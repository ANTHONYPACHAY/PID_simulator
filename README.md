# Simulador PID

## Installation

Esta aplicación requiere python >= 3.9.5.

Instala las siguientes dependencias.

```sh
pip install numpy
pip install tkinter
pip install ttkbootstrap
```

#### Ejecución

Ejecutar la aplicación con:

```sh
python interface.py
```
Interfaz de la aplicación en ejecución.
